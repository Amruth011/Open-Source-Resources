## AI & Machine Learning

### Learning Platforms
- [Fast.ai](https://www.fast.ai/) - Practical deep learning courses for coders
- [Andrew Ng's Machine Learning Course](https://www.coursera.org/learn/machine-learning) - Classic ML course on Coursera (audit for free)
- [Google's Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course) - Fast-paced introduction to ML
- [MIT Deep Learning](https://deeplearning.mit.edu/) - Free lectures and tutorials from MIT
- [Stanford CS229](https://cs229.stanford.edu/) - Machine Learning course materials

### Frameworks & Libraries
- [TensorFlow](https://www.tensorflow.org/) - End-to-end open source ML platform
- [PyTorch](https://pytorch.org/) - Open source ML framework
- [scikit-learn](https://scikit-learn.org/) - Machine learning in Python
- [Keras](https://keras.io/) - Deep learning API

### Resources
- [Papers with Code](https://paperswithcode.com/) - ML papers with code implementations
- [Distill.pub](https://distill.pub/) - Clear explanations of ML concepts
- [Machine Learning Mastery](https://machinelearningmastery.com/) - Practical ML tutorials

- [Jupyter Notebook](https://jupyter.org/) - Interactive computing environment

### Datasets
- [Kaggle Datasets](https://www.kaggle.com/datasets) - Large collection of free datasets
